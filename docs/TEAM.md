## How the project was built

IMMUTRACE was built using **AI-augmented development with Anthropic Claude Code**.
Manuel provided requirements, architectural decisions, testing strategy,
validation, and domain expertise; Claude Code generated code, tests, and
documentation under Manuel's direction. The result: a universal cryptographic
audit layer with **153 passing tests**, real AES-256-GCM encryption, Shamir
3-of-5 key custody, eIDAS-ready timestamping, and **live Polygon-mainnet
anchoring** (see `ARCHITECTURE.md`, `SECURITY_MODEL.md`, `BLOCKCHAIN_PROOFS.md`).
